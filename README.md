### 🚀 Overview
**MATHS HELP** is a sophisticated Java-based web application designed for precise geometric area calculations. It features a modern **Architectural Workspace** UI, providing a professional engineering environment for technical analysis.

Instead of a basic form, users interact with a **Clean Corporate Suite** dashboard that is responsive, accessibility-compliant, and production-ready.

## ✨ Key Features
- 🏢 **Professional Corporate UI** – Sleek, modern design with a focused "Workspace" layout.
- ⚡ **Precision Computing** – High-accuracy floating-point calculations for geometric analysis.
- 🐳 **Dockerized Deployment** – Multi-stage Docker build for professional cloud hosting.
- 📱 **Fully Responsive** – Optimized for seamless use across desktops, tablets, and mobile devices.
- 🔐 **Secure Architecture** – Stateless calculation logic ensuring high performance and security.

## ⚙️ Local Setup

1. **Clone the project**
   ```bash
   git clone https://github.com/ARYAN-SINGH-10/MathsHelp-AreaCalc.git
   ```

2. **Run via Docker (Recommended)**
   ```bash
   docker build -t maths-help .
   docker run -p 8080:8080 maths-help
   ```

3. **Manual Deployment**
   - Compile `Area1.java` using `servlet-api.jar`.
   - Place the `.class` file in `WEB-INF/classes`.
   - Deploy to any **Apache Tomcat 9.0+** server.

---

## 💡 How It Works
1. **User enters dimensions** into the professional Workspace interface.
2. **Java Servlet (Area1)** receives and validates the input parameters.
3. **The Dimension Engine** computes the area with high precision.
4. **The Results Dashboard** renders a technical report with full CSS integration.
